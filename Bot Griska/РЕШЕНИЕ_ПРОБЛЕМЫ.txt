❌ ПРОБЛЕМА:
───────────────────────────────────────────────────────────
У вас Python 3.14.1 - очень новая версия!
Некоторые библиотеки требуют компиляции, но не хватает:
- Microsoft Visual C++ Build Tools (для aiohttp)
- Rust (для pydantic-core)


✅ РЕШЕНИЕ 1 (РЕКОМЕНДУЕТСЯ):
───────────────────────────────────────────────────────────
Установите более новые версии библиотек без строгих версий:

pip install aiogram python-dotenv aiohttp

Это установит последние версии, которые могут иметь готовые
бинарные пакеты для Python 3.14.


✅ РЕШЕНИЕ 2:
───────────────────────────────────────────────────────────
Используйте Python 3.11 или 3.12 (более стабильные версии):

1. Скачайте Python 3.12 с https://www.python.org/downloads/
2. Установите с галочкой "Add to PATH"
3. Переустановите библиотеки


✅ РЕШЕНИЕ 3:
───────────────────────────────────────────────────────────
Установите Microsoft Visual C++ Build Tools:

1. Скачайте: https://visualstudio.microsoft.com/visual-cpp-build-tools/
2. Установите "C++ build tools"
3. Переустановите библиотеки


✅ РЕШЕНИЕ 4 (БЫСТРОЕ):
───────────────────────────────────────────────────────────
Попробуйте установить без компиляции (используя готовые wheels):

pip install --only-binary :all: aiogram python-dotenv aiohttp

Или попробуйте установить aiogram без строгих версий:

pip install aiogram python-dotenv "aiohttp>=3.9.0"
